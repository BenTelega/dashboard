# Static Site Skill

## Описание
Этот скилл предоставляет инструкции и рабочие процессы для управления статическим сайтом, развернутым с использованием Docker и Nginx.

## Структура проекта
- `index.html` — основная HTML-страница
- `Dockerfile` — конфигурация для сборки образа frontend
- `nginx.conf` — конфигурация reverse proxy
- `docker-compose.yml` — конфигурация сервисов frontend и nginx

## Рабочие процессы

### 1. Запуск сайта
Чтобы запустить сайт, выполните следующую команду:
```bash
cd ~/dashboard && docker compose up -d
```
Сайт будет доступен на `http://localhost:5001`.

### 2. Остановка сайта
Чтобы остановить сайт, выполните следующую команду:
```bash
cd ~/dashboard && docker compose down
```

### 3. Обновление контента
Чтобы обновить контент сайта:
1. Отредактируйте файл `index.html` с новым контентом.
2. Пересоберите и перезапустите контейнеры:
```bash
cd ~/dashboard && docker compose down && docker compose up -d --build
```

### 4. Просмотр логов
Чтобы просмотреть логи работы контейнеров, выполните:
```bash
cd ~/dashboard && docker compose logs -f
```

### 5. Проверка статуса
Чтобы проверить статус работающих контейнеров, выполните:
```bash
cd ~/dashboard && docker compose ps
```

## Примечания
- Убедитесь, что порты 80 и 5001 свободны перед запуском сайта.
- Для внесения изменений в конфигурацию Nginx отредактируйте файл `nginx.conf` и перезапустите контейнеры.
